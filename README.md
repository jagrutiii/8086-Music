# 8086 PC Speaker Music Generator

## Overview

This project implements a simple music generator using **8086 Assembly Language** and the **IBM PC Speaker**. Musical notes are generated by programming the **8253/8254 Programmable Interval Timer (PIT)** and controlling the speaker through direct hardware I/O ports.

The program plays the ascending and descending swaras of **Raag Malkauns** by storing note frequencies and durations in a memory-based song table and sequentially generating tones through the PC speaker.

---

## Features

* Written entirely in 8086 Assembly Language
* Direct hardware interaction using I/O ports
* Uses the 8253/8254 Programmable Interval Timer (PIT)
* Generates square-wave audio signals through the PC speaker
* Table-driven song representation using note-duration pairs
* Demonstrates low-level memory access, register manipulation, and hardware control

---

## Hardware Concepts Used

### Programmable Interval Timer (8253/8254)

The PIT receives a fixed clock input of:

```text
1,193,180 Hz
```

The output frequency is determined using:

```text
Frequency = 1193180 / Divisor
```

The program sends a 16-bit divisor value to PIT Channel 2, which generates the corresponding square wave.

### PC Speaker

The generated square wave is routed to the PC speaker through Port `61h`, allowing audible tone generation.

## Program Flow

```text
Initialize DS
        ↓
Load Song Table Address
        ↓
Read Note Divisor
        ↓
Read Duration
        ↓
Configure PIT
        ↓
Load Divisor
        ↓
Enable Speaker
        ↓
Delay for Duration
        ↓
Disable Speaker
        ↓
Repeat Until End Marker
```

---

## Song Representation

Each note is represented by two values:

```assembly
dw divisor, duration
```

Example:

```assembly
dw 4560,25
```

Where:

* `4560` = PIT frequency divisor
* `25` = note duration

The song table is traversed using the `SI` register.

| Swara     | Divisor | Frequency (Hz) |
| --------- | ------: | -------------: |
| Sa        |    4560 |         261.66 |
| Komal Ga  |    3800 |         313.99 |
| Ma        |    3421 |         348.78 |
| Komal Dha |    2850 |         418.66 |
| Komal Ni  |    2535 |         470.68 |
| Upper Sa  |    2280 |         523.32 |

The program plays:

```text
S g M d n S'
S' n d M g S
```

representing the ascending and descending scale of Raag Malkauns.
